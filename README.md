## 問題
PersonGroupクラスの
- addGroupMember(Person person)
- contains(Person person)
- contains(int personId)
を実装してください。

## クラスについて
PersonGroupクラスは人のグループを表すクラスです。  
以下の情報を保持します。
- グループID
- グループ名
- グループのメンバー

また、以下の機能を持っています。
- そのグループに特定の人を追加する
- そのグループに特定の人が含まれているか確認する
- そのグループに特定の個人番号を持つが人が含まれているか確認する
