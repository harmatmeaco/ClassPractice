## 問題
Organizationクラスはグループ(PersonGroup)をまとめるクラスです。  
以下のメソッドを実装してください。

- Organizationにグループを追加するメソッド
- Organizationにグループのリストを追加するメソッド
- グループに所属するメンバーの人数を配列で返却するメソッド
- グループに所属するメンバーの人数をリストで返却するメソッド
